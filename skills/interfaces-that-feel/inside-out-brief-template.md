## Material Implications

*Derive these from the recipe above. Don't fill them in from memory — derive them.*

**Timing**
→ When does the response begin? (at gesture start / at commit / at completion)
→ Duration range:
→ Is there a pause before response? If so, what does it communicate?

**Dynamics**
→ Spring: stiffness __ damping __ mass __
→ Overdamped / critically damped / underdamped — which and why?
→ Does gesture velocity carry through into the animation?

**Modality**
→ Haptic: which type, which intensity, at which moment?
→ Visual: scale / opacity / elevation change?
→ Is the modality response in the same register as the action?

**Expression**
→ Does the response vary based on how the action was performed?
→ If not, should it?
