# interfaces that feel

A skill for Claude Code based on Marie Spreitzer's design philosophy.

Builds and reviews interfaces through one lens: does this feel like it was made by a human who cared about how you'd feel using it?

## Install

```bash
npx skills add mariespreitzer/skill
```

## What it covers

- The Feel Framework — three questions before any product decision
- Motion principles with emotional framing
- Copy voice — error messages, empty states, loading states
- Emotional timing — when to feel vs when to stay out of the way
- Before/After/Why review table for UI code

## Reference aesthetic

How We Feel, Headspace, Gentler Streak, Amie, Arc Browser.
